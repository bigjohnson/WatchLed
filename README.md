# WatchLed
Firebox 500/750/100 Watchguard led board with arduino

Chip order:
1. Control Disarm, Armed, Sys_A, Sys_B leds
2. Control triangle leds
3. Control triangle leds
4. Control Traffic bar leds
5. Control Load bar leds
